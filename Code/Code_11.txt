Глава 11. Логические вентили.
В 20 веке Клод Шеннон сделал открытие, которое утверждало, что логические выражения можно реализовать при помощи электрических цепей. Преимуществом этого открытия являлось то, что для упрощения схемы можно было сначала упростить логическое выражение, а по нему уже построить цепь. 
Для управления переключателями при помощи электричества используют логические вентили и, в частности - реле. Логический вентиль - соеденение реле последовательным или параллельным способом для решения простых задач. Существует 4 вида логических вентилей: И, ИЛИ, ИЛИ-НЕ, И-НЕ.  Также существуют буфер, который усиливает сигнал и инвертор, который инвертирует сигнал(0 => 1, 1 => 0). Выход одного вентиля может являться входом одного или нескольких других вентелей, но несколько вентелей не могут быть подключены к одному. 
Дешифратор позволяет при помощи n бит представить n^2 разных значения.
