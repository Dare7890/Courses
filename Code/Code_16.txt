Глава 16. Сборка памяти.
Для того, чтобы хранить 1 бит информации в память используем D-триггер, который при подаче 1 в виде сигнала на вход "Clk", запоминает сигнал, который был подан на вход "D". Для того, чтобы сохранить 1 байт информации, соединим 8 таких триггеров последовательно, где входы "Clk" соединены одной шиной между собой, а вход под данные у каждого свой. При подаче 1 на вход "Clk" значение запишется в импровизированную память.
Чтобы создать память с произвольным доступом необходимо, чтобы каждый бит отдельно записывался в выбранный нами триггер и чтобы мы могли читать данные также из отдельгых триггеров. Для этого необходимо: 
-несколько переключателей, которые позволяют задать номер выхода, на который мы будем записывать значение и считывать в последствии(адреса);
-дешифратор, при помощи которого сигнал записи будет подаваться именно на вход того триггера, который выбраг переключателями;
-D-триггеры, входы данных которых соединены одной шиной, а входы записи подключены каждый отдельно к дешифратору;
-селектор, при помощи которого мы считываем из выхода, который мы задали также при помощи переключателей, информацию.  

Дешифратор позволяет из n значений получить 2^n различных вариантов, причем на только на одном выходе сигнал будет равен 1. 
Селектор позволяет из нескольких вариантов выбрать один.

Адрес определяет, какая защелка соханит сигнал "Данные" под воздействием сигнала "Запись". Также, существует память с последовательным доступам, где для того, чтобы считать значение по адресу 101 сначала надо считать значение с адреса 100.

Если к двум Массивам RAM 8x1 подвести общую шину ко входу "Адрес", то полчится массив 8x2, то есть 8 значений по 2 бита, но если к ним также поставить дешифратор и селектор, которые позволяют выбрать в какой массив RAM будет производиться запись, то получится массив 16x1.

Так как триггеры состоят из реле, основу которых составляют магниты, то при отключении питания электромагниты потеряют свои свойства и, как следствие, данные из памяти потеряются. Поэтому RAM называют энергозависимой.
