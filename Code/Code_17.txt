Глава 17. Автоматизация.
Цифровой компьютер состоит из четырех основных частей: процессор, устройство ввода/вывода и памяти. Устройство ввода/вывода может быть представлено в виде переключателей и лампочек, память в виде защелок и дешифратора с селектором, а процессор состоит из аккумулятора (защелка, которая хранит сначала начальное значение, а потом результаты арифметических операций над этим значением), инвертора и сумматора(АЛУ - арифметико-логическое устройство). Компьютерная программа состоит из команд, которые состоят из трех байтов, которые в свою очередь хранятся в памяти в виде кода операции - 1 байт(сложение, загрузка, сохранение) + адрес памяти - 2 байта(указывает откуда надо загрузить значение, куда сохранить). АЛУ производит серию действий в соответсвии с кодом команды. Данные и коды операций также хранятся в виде шестнадцатиричного кода в памяти RAM. Помимо арифметических операций существуют такие операции, как: сложить с переносом, вычесть с заимствованием, безусловные и условные переходы(позволяют идти по ячейкам памяти, в которых хранится код программы, не последовательно, а с переходом на определенный участок памяти). Для хранения 16битных чисел в 8разрядной памяти можно разбить значение на 2 разряда - старший и младший, которые будут хранится в смежных ячейках памяти. Код программы представляет собой машинный код. Существует также язык ассемблера. Он помогает описывать действие машинного кода при помощи мнемоник.
