Глава 1. Лучшие друзья.
Фонарик - как система передачи информации между людьми. Аналог кода, который позволяет осуществлять коммуникацию. "Код" используется повсеместно в процессе общения: речь, печатные символы. Азбука  Морзе во взаимодействии с фонариком также является кодом, где долгое включение фонарика отвечает за передачу тире, а короткое включчение - точку. Получатель должен расшифровать полученный шифр при помощи специальной таблицы. Таким образом можно наладить передачу сообщений на дальнее расстояние между людьми.
