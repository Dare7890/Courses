Глава 22. Операционная система. 
При запуске компьютера МП начинает считывать с ОП машинные команды, но так как RAM при отключении электропитания теряет всю информации, вместо машинных команд будет записана последовательность из случайных значений. 
Файловая система - способ хранения данных, при котором они организуются в файлы. Файловая сисетма входит в сотав операционной системы. Одной из первых ОС была CP/M. В ней каждый файл имел свое имя и файлы могли хранится в несмежных секторах. Сектора на дорожках входили в состав в блок выделения памяти, где каждому блоку соответствовало 8 дорожек. Первые 2 блока были заняты каталогом, в котором хранилась вся информация о существующих файлах. Сама ОС содержалась на первых двух дорожках. Изначально ОС записывалась в память с диска. В ПЗУ хранится загрузчик программы, который считывал первый сектор в память и запускал его. Этот сектор содержал код для загрузки всех остальных секторов в память. Этот процесс называется загрузкой системы. Также, ОС предоставляет программам доступ к аппаратным средствам, которые подключены к компьютеру. В современных ОС вместо CP/m используется таблица размещения файлов. Дисковое пространство делится на кластеры. Каждый файл состоит из нескольких кластеров. В каталоге, в котором находится файл, содержится только начальный кластер файла, а в таблице FAT указывается для каждого кластера на диске где находится продолжение файла. Система уже содержится в ПЗУ компьютера.
