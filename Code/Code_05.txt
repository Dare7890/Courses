Глава 5. Заглядывая за угол.
Для построения двухстороннего телеграфа можно использовать следующие элементы: 2 лампочки, 2 батарейки, 2 переключателя и провода. При правильно построенной схеме с общим проводом, лампочки будут загораться как при нажатии одного из человека, так и двумя людьми. Для экономии провода можно воспользоваться заземлением в виде хорошего проводника с обширной поверхностью, закопанным в землю. При подведении провода к такому проводнику происходит передача электронов через землю. Земля обладает высоким сопротивлением. Минус данного телеграфа в том, что при большом расстоянии между источником и лампочкой сила тока будет мала и, как следствие, лампочка будет тускло гореть, если вообще загорится. Как вывод: телеграфный провод невозможно протянуть на неограниченное расстояние.
