Глава 9. За битом бит.
При помощи набора битов можно закодировать любую информацию. Бит - мельчайший фрагмент информации. Каждый возможный вариант можно закодировать определенным набором битов. Количество битов определяет сколько кодов можно получить с их помощью. Также работает обратная операция, которая сводится к вычислению логарифма от количества кодов по основанию 2. Кодирование определенного товара в магазине также осуществляется при помощи определенных кодов, которые формируются из битов. Каждая черная полоса соответствует 1, белая - 0. При помощи таких комбинаций можно закодировать любой продукт. Код состоит из типа (обычный товар, акция), кода производителя, кода продукции и остатка. Код можно считывать в обоих направлениях. 
Для представления любой информации в виде битов достаточно пересчитать количество возможных вариантов и присвоить каждому из вариантов свой ключ.
