Глава 14. Обратная связь и триггеры. 
Электричество приводит в движение предметы. Для построения зуммера используется осциллятор. Значение осциллятора чередуется между 0 и 1 с течением времени. Его также называют часами, так как по количеству колебаний можно посчитать пройденное время. Триггер позволяет запомнить бит информации. Это осуществляется за счет подключения вентилей обратной связью, то есть выход следующего вентиля соединен со входом более "младшего" вентеля в цепи. Существует 4 основных вида триггера: RS, D, JK и T-триггеры.
RS-триггер (Reset/Set) устанавливает выход Q в 1 при подаче 1 на вход S, при подаче 1 на вход R выход Q становится равным 0. При подаче и на R, и на S сигнала 0, сигнал на выходе Q не изменяется. Комбинация в виде подачи на входы R и S сигналов 1 является запрещенной. Входами такого триггера являются R и S, выходами - Q и not(Q).
D-триггер делится на триггер, который срабатывает по уровню и по фронту. 
D-триггер по уроню имеет входы D (Data) и Clk(Clock). При Clk = 0 выходы Q и not(Q) остаются неизменными, то есть сохраняется прошлое значение, при Clk = 1, выход Q становится равным входу D, что позволяет перезаписать значение. Такой триггер также называется защелкой, так как позволяет сохранить один бит данных для дальнейшего использования. Если соединить несколько таких защелок последовательно, то можно, например, записать значения из сумматора в триггеры, тем самым сохранив значение. Может существовать бит Clr(Clear), который позволяет очистить выход Q( Q = 0)
Селектор 2 на 1 при помощи инвертора позволяет выбрать один из двух возможных вариантов продолжения движения по цепи.
D-триггер по фронту позволяет перезаписать значение на выход Q при изменении сигнала на входе Clk с 0 на 1. При подключении в D-триггере по фронту выхода Q с входом триггера D, такачя конструкция будет называться делителем частоты. При соединении нескольких таких триггеров можно построить счетчик.
JK-триггеры (Jerk/Kill) отличается от RS только тем, что доступна комбинация J=K=1. При таком варианте выход Q меняет свой сигнал.
T-Триггер (Toggle/ счетный триггер). Позволяет инвертировать входной сигнал.
