Глава 19. Два классических микропроцессора.
Процесс считывания команды из памяти называется выборкой команд. Команда в микропроцессоре I8008 может состоять из 1 - 3 байт. Также существуют аккумулятор и 6 дополнительных регистров для хранения восьмибитных значений: A, B, C, D, E, H, L. H и L часто используются для адресации памяти. 
LDA A, [0018] - пример прямой адресации
MOV B, [HL] - индексная адресация
MVI C, 37h - непосредственная адресация

В микропроцессоре также существует система флагов: флаг переноса, нуля, знак, четности и вспомогательного переноса. Они хранятся в регистре, который называется словом состояния программы. Существуют три сдвига: логический, арифметический и циклический. 
Стек - участок памяти, работающий по принципу LIFO. Для обращения к нему используется 16битный регистр - указатель стека. При записи значения в стек сначала уменьшается значения адреса памяти , затем по этому адресу записывается значение из регистра. Pc - счетчик команд. Показывает из какого адреса будет выполнятся следующая команда. При переходе к подпрограмм старый адрес команды записывается в стек. Переферийные устройства - все подключеннык к МП устройства кроме памяти. Чтобы переферийное устройство привлекло внимание МП используются прерывания - сигнал от переферийного устройства на вход INT. Адрес старой команды записыватся в стек, а МП переходит по новому адресу команды, который задается прерыванием. 
Кэш - массив сверхскоростной оперативной памяти внутри процессора, в котором хранятся недавно выполненные команды.
