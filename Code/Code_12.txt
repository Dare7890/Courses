Глава 12. Двоичный сумматор.
Для того, чтобы компьютер мог выполнять любые арифметические операции достаточно реализовать алгоритм сложения. Для этого необходимо собрать сумматор. На вход сумматора поступают числа в двоичной СС. Для операции сложения сумматор состоит из трех входов: вход переноса, вход A и вход B. Вход переноса необходим для отслеживания факта, было ли переполнение при сложении прошлого разряда или нет, последние два - для подачи битов, которые необходимо сложить. Сумматор состоит из двух полусумматоров и вентиля И. На первый полуссуматор поступают 2 бита для сложения, на второй поступает результат сложения первого сумматора и бит разряда с прошлой итрерации сложения. Результатом вычисления второго полусумматора является сложение двух битов. Биты переносов с первого и второго полусумматоров поступают на вентиль И и образуют бит переноса для всего сумматора. Полусумматор стостоит из вентиля И и вентиля Искл-ИЛИ.
