Глава 2. Коды и комбинации.
Код является двоичным, если любой его элемент может содержать только одно из двух возможных значений. Пример: азбука Морзе, где возможные значения - точка и тире. Формула, для вычисления количества уникального кода алфавита: Math.Pow(2, количество разрядов (в примере это количество точек и тире)). Для расшифровки последовательного кода удобно пользоваться древовидной схемой. Для этого необходимо проодить по вершинам дерева слева направо.
