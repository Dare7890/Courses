Глава 23. Фиксированная точка, плавающая точка.
Для представления дробного числа с фиксированной точкой можно записать число в формате BCD, где первые 4 или 8 бит отводятся под знак, чтобы было выделено целое число байтов. 
Например: -165,25 = 00010001 11001001 00100101 = 11h 65h 25h.
Но для такого формата необходимо заранее сообщить программе в каком месте находится разделитель. ТО есть уместно использовать данный формат в тех случаях, где мы будем заранее уверены, что не надо будет увеличивать количество десятичных знаков. 
В противном случае используется формат с плавающей точкой. В нем число записывается следующим образом 
4900000 = 4.9 * 10^6, где
4.9 - это дробная часть или мантисса
6 - это порядок, то есть на сколько будет сдвинут разделитель в значащей части
Значащая часть должа быть нормализована, то есть значение должно быть в диапазоне от 1 до 10. Существуют 2 формата: число одинарной точности (4 байта на значение) и двойной точности (8 байтов). Одинарный формат состоит из 1 бита под знак, 8 бит под порядок и 23 - для дробной части числа. В двойном формате 11 бит под порядок и 52 для дробной части. 
Для вычисления значения с плавающей точкой используется математический сопроцессор, который работает только в паре с МП.
