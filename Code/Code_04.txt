Глава 4. Устройство фанарика.
Рассмотрение электричества сковзь призму фонарика. Фонарик можно собрать из следующих предметов: выключатель, батарейка, провода и лампочка. Электричество возникает в процессе вышибания электрона из атома. При возникновении дисбаланса электронов происходит  автоматическое их сбалансирование, в результате чего возникает искра статического электричества. Электричесво в цепи - переход электронов от одного атома к другому. Для запуска этого явления необходим катализатор, например, батарейка. За счет химического вещества, которое образует на одном полюсе батарейке (катод) избыток электронов, а на другом (анод) - дефицит, электроны переходят от - батарейки к +.
