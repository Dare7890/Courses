Глава 13. А как насчет вычитания?
В компьютере вычитание произовдится при помощи арифметической операции сложения. Для того, чтобы вычитать числа без заимствования необходимо:
a) Если уменьшаемое больше вычитаемого: 
- вычитаемое инвертируется, добавляется 1, добавляется уменьшаемое и вычитается 1 из старшего разряда
б) Если уменьшаемое больше вычитаемого:
-вычитаемое инвертируется, складывается с уменьшаемым и результат ивертируется.
Для построения такой схемы понадобится сигнал инверсии, который пдключается к каждому входу логического вентиля Искл-ИЛИ. Таким обраом, если сигнал инверсии 1, то значение каждого бита числа инвертируется, если 0, то число остается неизменным.
Если количество разрядов чисел нам заранее известно, то можно распределить значения так, что старший бит числа отвечает за знак. Если 1 - то число отрицательное, 0 - положительное. Для представления числа -5 в двочной системе переводим 5 в двочный код, потом инвертируем и добавляем 1.
5 = 0000 0101
Инверсия = 1111 1010 + 1 = 1111 1011, тк старший разряд 1, то число отрицательное.
По самим числам невозможно понять, являются они отрицательными или нет. Сначала надо узнать, со знаком ли число или нет?

Вычитание 3 - 25 = -22
25 = 0001 1001; 3 = 0000 0011
-25 = 1110 0110 + 1 = 1110 0111

0000 0011 + 1110 0111 = 1110 1010 = -22
1110 1010 = 0001 0101 + 1 = 0001 0110 = 22
