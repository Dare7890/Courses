Глава 15. Байты и шестандцатеричные числа. 
Последовательность 8 битов составляют байт. Они составляют именно 8 разрядов, так как при помощи них легко хранить текст (большая часть языков содержит меньше 256 символов), оттенки цвета и т.д. 4 бита называются тетрадой. Один байт состоит из двух тетрад. Шестнадцатиричная СС состоит из следующих символов: 1-9, A-F, где A = 10, B = 11... С помощью шестнадцатиричной СС обозначают MAC и IP адреса. То, что число является шестнадцатиричным, тожно понять по латинской букве h в конце числа. 

52 = 0011_0100 = 34h
